# Charte Graphique — GameVault

Ce document récapitule les choix visuels effectués pour l'application GameVault, à inclure dans le **Chapitre 2** du Dossier Projet.

## 1. Univers Visuel
- **Style :** Dark Mode Premium, Gaming, Moderne.
- **Concept :** Glassmorphism léger, néons et contrastes profonds.

## 2. Palette de Couleurs
| Rôle | Couleur | Code Hex | Usage |
| :--- | :--- | :--- | :--- |
| **Fond Principal** | Deep Black | `#0B0E11` | Arrière-plan de l'application |
| **Fond Secondaire** | Dark Grey | `#15191C` | Cartes, Sidebar, Modales |
| **Accent Primary** | Electric Violet | `#8B5CF6` | Boutons principaux, liens actifs |
| **Accent Secondary** | Neon Blue | `#00D1FF` | Badges, indicateurs de statut |
| **Texte Principal** | Pure White | `#FFFFFF` | Titres et textes importants |
| **Texte Secondaire** | Light Grey | `#94A3B8` | Descriptions, labels, footer |
| **Erreur / Danger** | Vivid Red | `#EF4444` | Boutons de suppression, erreurs |

## 3. Typographie
- **Police de titres :** `Outfit` ou `Inter` (ExtraBold pour H1, Bold pour H2).
- **Police de corps :** `Inter` (Medium/Regular).
- **Tailles :**
    - H1 : 32px
    - H2 : 24px
    - Body : 16px
    - Caption : 12px

## 4. Iconographie
- **Bibliothèque :** Lucide Icons ou Phosphor Icons.
- **Style :** Outline (traits fins de 1.5px ou 2px) pour un look technologique.

## 5. Composants UI Clés
- **Boutons :** Radius de 8px, padding `12px 24px`. Effet hover avec légère augmentation de luminosité.
- **Cartes (Games) :** Aspect "Glass" (opacité 70%), bordure fine de 1px en `#1E293B`.
- **Inputs :** Fond sombre, bordure accentuée en Violet lors du focus.
